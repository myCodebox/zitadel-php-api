Get.php : 12

Get a session and all its information like the time of the user or password verification.

If no data is sent back from the Zitadel API for e.g. 2FA tokens, an error will occur when trying to access the data. Some functions return a date in the format YYYY-MM-DDThh:mm:ss.fZ e.g. 2024-04-08T14:37:09.846600Z.
